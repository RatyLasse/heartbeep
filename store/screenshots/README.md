# Play Store Screenshots

Place your Play Store screenshots here before submission.

## Requirements
- **Format:** PNG or JPEG
- **Minimum:** 2 screenshots required; recommended 4–8
- **Dimensions:** minimum 320px on either side, max 3840px, max 2:1 aspect ratio
- **Device:** Phone screenshots (required); tablet optional

## Suggested shots
1. Main screen — device connected, heart rate live
2. Active monitoring session with BPM alert configured
3. Session history list
4. Session detail with HR graph
